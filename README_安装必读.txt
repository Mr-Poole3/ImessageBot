⚠️ ImessageBot 安装与使用必读 ⚠️

由于本程序是为个人定制开发，未经过 Apple 官方公证，在第一次安装打开时，系统会弹出安全警告。请按照以下步骤操作：

1. 第一次打开报错：
   - 将 ImessageBot 拖入“应用程序”文件夹后。
   - 尝试打开时，如果提示“无法验证开发者”或“已损坏”，请点击取消。
   - 进入“系统设置” -> “隐私与安全性”。
   - 向下滚动找到“安全性”部分，点击【仍要打开】按钮。

2. 授予必要权限（非常重要）：
   - 由于本程序需要读取 iMessage 聊天记录，必须手动开启权限。
   - 进入“系统设置” -> “隐私与安全性” -> “完全磁盘访问权限”。
   - 点击左下角的“+”号，或者直接将 /Applications/ImessageBot.app 拖入列表并确保勾选开启。

3. 配置：
   - 程序运行后，配置文件保存在用户目录下的 ~/.imessagebot 文件中。

祝使用愉快！
